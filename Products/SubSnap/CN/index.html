隐私协议

更新日期：2026年1月28日

1. 适用范围

本隐私政策适用于 SubSnap 应用及相关服务。我们作为个人开发者运营本服务， 并遵守《个人信息保护法》《网络安全法》《数据安全法》等适用法律法规。

在使用服务前，请仔细阅读本政策。若您不同意本政策，请停止使用服务。 您继续使用服务即视为已阅读并同意本政策。

2. 我们处理的信息及目的

2.1 订阅管理与统计

  • 订阅名称、服务提供方、金额、币种、周期、续费日期、分类、支付方式等
  • 用途：展示和管理订阅、续费提醒、统计分析

2.2 OCR 图片识别

  • 您主动上传的订阅确认页、账单或收据截图/照片
  • 用途：识别并提取订阅信息，辅助自动填写

2.3 通知、购买验证与备份恢复

  • 提醒计划、iOS 推送 token、App Store 收据信息
  • 订阅与提醒数据快照，以及随机生成的设备 ID

2.4 运维与安全

  • 请求 ID、时间戳、响应状态码等基础日志，以及必要网络元数据
  • 用途：运行维护、安全审计、故障排查和防滥用

我们不使用第三方行为追踪或广告分析 SDK，也不会将您的信息用于广告投放。

3. 敏感信息、共享与委托处理

OCR 图片可能包含账单或付款信息，属于可能涉及财务信息的敏感个人信息。 仅在您主动使用 OCR 功能时处理。

  • OCR 服务商：Ark API / OpenRouter，仅用于图片识别
  • 云基础设施：用于后端服务和数据存储，例如 AWS 或国内云服务
  • Apple 服务：用于 App Store 购买验证和 iCloud 备份

我们不会出售个人信息，也不会在未经授权的情况下公开披露。 仅在法律要求、保护合法权益或防止欺诈和安全事件时共享必要信息。

4. 存储、保护与跨境说明

  • 本地数据保存在您的设备中，主要使用 SQLite 存储。
  • 服务器快照用于备份与恢复，可按配置启用加密存储。
  • 我们使用 HTTPS/TLS、访问控制、最小权限原则等安全措施。
  • 本地数据在您主动删除前保留；服务器快照会被新快照覆盖，或按请求删除。
  • 当前服务不计划进行个人信息出境；如未来涉及跨境传输，将依法履行告知和同意义务。

5. 您的权利

您可以访问、更正、删除个人信息，撤回同意，获取副本，或在适用情形下限制处理。

您可以通过应用内功能或邮件联系我们行使上述权利，我们会在合理期限内处理您的请求。
